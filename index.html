GGs Xbox Game Bar.

Ghost is a widget for the Xbox Game Bar on Windows 10. At any time, press + G to bring up the overlay.

Heads up display.

Pin Ghost to have it overlay on top of Destiny 2 to monitor your bounties and triumphs as you play.

Finish your Triumphs.

Ghost can track Triumphs, as well as Catalysts and Quests. You can't do that in-game (for now).

FAQ.

Q: Is this legit? Will I get banned for using this?

Yes, this is ✨ legit ✨. No, you will not get banned. This uses the official Xbox Game Bar SDK from Microsoft, a part of the Windows operating system.

Q: How often does this update?

Ghost checks with the Destiny API every 30 seconds to get your latest progress, however the Destiny API may send slightly older data sometimes to keep the server running smoothly for everyone. The "Updated at" time in Ghost will tell you the last time it got new, fresh data.

Known issues.

If you're having issues with Ghost, make sure you're running at least Windows 10 version 1903 or 1909, with all installed Windows updates. Additionally, check the Microsoft Store for any updates for Xbox Game Bar.

Ghost crashes immediately when trying to log in.

While I investigate this issue, you can use this login URL to login with your Browser and you'll be redirected to Ghost.

Ghost gets stuck on "Loading Profile...".

A few users who've had this problem have reported that enabling cross-save on Bungie.net allows the profile to load.

Triumphs keep becoming untracked

A fix for this has been submitted to Microsoft and is awaiting certification. Until then, closing the Settings view while the profile updates should avoid this issue.
